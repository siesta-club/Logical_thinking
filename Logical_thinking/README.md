# MIND READER — 心理探偵

> 「なぜこの人はこう動くのか」を2分で読み切れ

## ゲーム概要

職場の人物4人の行動パターンを読み、その裏にある心理構造を分析する心理推理ゲームです。

各ケースに制限時間2分。表面的な行動ではなく「なぜそう行動するのか」という本質を選択肢から選びます。終了後は正誤だけでなく、あなたの強みと改善点を分析したフィードバックが表示されます。

## 作った経緯

ADHDの過集中という特性を、論理的思考とメタ認知のトレーニングによってプラスに転換してきた経験があります。

その過程で気づいたのは、「他者の行動を構造として読む力」が思考力の核心の一つだということ。この力はゲームという形で楽しみながら鍛えられるのではないかと思い、作りました。

## 遊び方

1. ページを開く
2. GAME START を押す
3. 人物の行動シナリオと証拠を読む
4. 2分以内に本質的な理由を選択して ANALYZE
5. 4ケースのフィードバックで自分の思考パターンを確認する

## 技術

HTML / CSS / JavaScript のみで動作します。外部ライブラリ・サーバー不要。
