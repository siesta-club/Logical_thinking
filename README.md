# 論理的思考トレーニングゲーム

> 「なぜそうなるのか」を読み切る力を鍛える

## 概要

論理的思考・メタ認知能力をゲーム形式で鍛えるシリーズです。各問題に制限時間2分。表面的な現象ではなく「なぜそうなるのか」という本質を選択肢から選びます。終了後は正誤だけでなく、あなたの強みと改善点を分析したフィードバックが表示されます。

## 作った経緯

社会を生きていく中で論理的思考トレーニングが如何に有用かを知り、「構造として読む力」をゲームで楽しみながら鍛えられないかと思い、作りました。

## 技術

HTML / CSS / JavaScript のみで動作します。外部ライブラリ・サーバー不要。
